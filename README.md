# TicTacToe Game

A minimalistic Java implementation of well known game to be played in console.

## How to run

```
java -jar game.jar
```

## How to play

```
 | |
-+-+-
 | |
-+-+-
 | |
 
Choose tile to place X (1-9): 5

 | |
-+-+-
 |X|
-+-+-
 |O|
 
Choose tile to place X (1-9): 6

 | |O
-+-+-
 |X|X
-+-+-
 |O|
 
Choose tile to place X (1-9): 4

 | |O
-+-+-
X|X|X
-+-+-
 |O|
 
Congratulation! You won.

```
